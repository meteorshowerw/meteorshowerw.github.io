Plugable USB-C 変換グラフィックアダプタ、ディスプレイ変換 Windows 用 デュアル 4K HDMI および DisplayPort、イーサネットポート、USB 3.0、USB Type-C 対応



説明文



・2 画面 4K モニター対応 - HDMI または DisplayPort 対応 4K モニタを同時に最大 2 台接続できます。



最大解像度 3840x2160 @ 60Hz で、1920x1080 (1080p) などの一般的な解像度にも対応しています。（4K 解像度の外部モニターは、システム負荷を考慮して２台までにすることをお勧めします。）



・柔軟な接続 - HDMI と DisplayPort の組み合わせが自由にできます（HDMI x 1 と DP x 1、HDMI x 2、など）。PC 側のポートタイプに応じて USB 3.0 と USB Type-C どちらにも接続できるよう変換アダプターが同梱されています。このアダプターのイーサネットポートを使うと、WiFi 接続よりも安定した有線 LAN 接続が簡単にできます。



最大 1 Gbps（10/100/1000 Mbps）スピードに対応しています。



・互換性 - Windows 11、10、8.1 および macOS 10.15以上（11 以上を推奨）のラップトップ、タブレットなどで使用することができます。このアダプタを接続する前に、Windows Update は最新状態へ更新することをお勧めします。Linux、ChromeOS システムでは使用できません。Mac システムに関する注意点： macOS に正しいバージョンの DisplayLink ドライバを導入する必要があります。
