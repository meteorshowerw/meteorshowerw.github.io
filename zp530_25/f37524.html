アンティークトランクケース(木製)

1975年に日立製作所で開発されたミニコンピューター「HITAC 10IIシステム」で使うプログラムを書き込んだ「紙製テープ」の保管ケースです。

(希少品)

私は譲りうけた後少しの間文房具などの収納兼インテリアに使っていましたが現在は日焼けなどしないように保管中です。(写真3枚目はインテリア使用時を再現)　コミック本も入るので屋外フリマでの古書販売での展示などにも使えそうです。

(サイズ)

・内寸:478mm x 339mm x 深さ90mm、フタ部は深さが31mm

・外寸:497mm x 355mm x 高さ137mm（金具類は含まず)

・中仕切りは2.4mmベニヤ板で、

　59mm x 474mm_溝3本 x9枚と59mm x 318mm_溝9本 x3枚を組み合わせます。(写真2枚目と写真13枚目ご参照)

・仕切られた区画のサイズは64mm x 116mm(仕切り板の板厚除く)

・仕切り板の枚数を加減すると区画サイズはある程度自由に変えられます。

(仕切りの構成)

・ケース内壁に2.4mm x 59mm x 476mmを2枚と59mm x 328mm(溝あり)2枚を置きます。(写真12枚目ご参照)

・この溝に写真13枚目にある溝付きの長い板をはめこみ、その後同写真13枚目の溝付きの短い方の板を長い板の溝にはめ込みます。

(その他)

・フタ裏面のコンピュータープログラムテープの識別表示は紙を糊付けしてあるようですのではがさない方がよいかと思います。若干のシワがあります。

・ペットなし、喫煙者なしです。
